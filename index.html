Primary Color
Dark Gray

Default Background Color, when background color is light, this should be the primary font color.
Secondary Color
Denim
Hex: #235db8

Default Heading and Sub-Heading color, to bbe used for links on light background.
Tertiary Color
Light Blue
Hex: #8ab4f8

Accent color, to be used for links on dark background.
Font Color
Light Grey
Hex: #cfcece

Primary Font Color, to be used for links on dark background.
